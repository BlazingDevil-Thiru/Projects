# PayTrack Pro 💰

A sleek, lightweight, and privacy-focused daily payment tracker built with Vanilla JS and Tailwind CSS. Track daily expenditures, view monthly summaries, and navigate through payment history with ease.

## 🚀 Features
- **Dynamic Calendar:** Full navigation between months/years.
- **Smart Color Coding:** Automatic UI updates based on payment thresholds.
- **Live Analytics:** Real-time calculation of total spend and monthly averages.
- **Local Persistence:** Data is saved locally in your browser (no database required).
- **Fully Responsive:** Optimized for both Desktop and Mobile.

## 🛠️ Tech Stack
- **Frontend:** HTML5, JavaScript (ES6+)
- **Styling:** Tailwind CSS (via CDN)
- **Icons:** FontAwesome
- **Storage:** Browser LocalStorage

## 📖 How to Use
1. **Set Amount:** Enter the payment value in the top input field.
2. **Log Payment:** Click on any date in the calendar to assign that amount.
3. **Navigate:** Use the arrows at the top to switch between months.
4. **Wipe Data:** Use the "Clear Records" button to reset your local database.

## 📦 Deployment
This project is a single-page application (SPA). To deploy:
1. Push this code to a GitHub repository.
2. Connect it to **Netlify** or **Vercel**.
3. It will be live in seconds!
